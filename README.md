Rec - Catálogo Profesional de Audio y Video 📹🎙️
Rec es una aplicación nativa para Android desarrollada en Kotlin, diseñada para la gestión y visualización de equipos profesionales de producción audiovisual. Este proyecto forma parte de un entorno académico enfocado en el desarrollo de interfaces modernas, navegación entre fragmentos y gestión de datos en tiempo real.

🚀 Características
Catálogo Dinámico: Visualización de equipos de alta gama (cámaras, grabadoras, micrófonos) mediante RecyclerView.

Navegación Fluida: Implementación de Fragments para una experiencia de usuario sin interrupciones entre Inicio, Catálogo, Favoritos y Carrito.

Sistema de Selección: Capacidad de enviar productos específicos del catálogo al carrito de compras utilizando Bundles y paso de parámetros.

Diseño Profesional: Interfaz limpia basada en una paleta de colores corporativa (Rojo Coral y Gris Fondo).

🛠️ Stack Tecnológico
Lenguaje: Kotlin.

Entorno: Android Studio.

UI Layout: XML con componentes de Material Design.

Arquitectura: Basada en Fragments y Adaptadores personalizados para la gestión de listas.

📁 Estructura del Proyecto
El código está organizado siguiendo las mejores prácticas de empaquetado por funciones:

auth: Gestión de Login y Registro.

inicio: Pantallas de Splash y Bienvenida.

main.productos: Núcleo de la app (Catalogo, Carrito y Adaptadores).

productos: Clases de modelo de datos.

🔧 Instalación
Clona este repositorio:

Bash
git clone https://github.com/tu-usuario/Rec.git
Abre el proyecto en Android Studio.

Sincroniza el proyecto con los archivos de Gradle.

Ejecuta la aplicación en un emulador o dispositivo físico con Android 8.0 (API 26) o superior.
